
# REALEST FASHION STORE API

A RESTful API application for a fashion website.

## Tech Stack

**Client:** React, MaterialUI, NextUI

**Server:** Java, Spring Boot


## Features

- Login with third party
- Cart management
- Online payment
- Review product
- System management
